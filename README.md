## ğŸ“š Physics

The simulation is based on Snell's Law:

$$
\frac{\sin \alpha_1}{\sin \alpha_2} = \frac{n_2}{n_1}
$$

Where:
*   $n_1, n_2$ = Refractive indices of the media
*   $\alpha_1$ = Angle of incidence
*   $\alpha_2$ = Angle of refraction
